How the palette works. One brand ramp + one semantic set, re-tuned per theme. In dark mode, semantics live at ~80% lightness with low chroma; in light mode, the same hues drop to ~40% lightness with higher chroma. That keeps hue identity consistent across themes while passing contrast in both — so a "green pill" still reads as the same color concept whether you're on black or white.
